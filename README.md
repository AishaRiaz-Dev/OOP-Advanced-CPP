# 💻 Advanced Object-Oriented Programming (C++)

This repository showcases a collection of systems designed to demonstrate mastery of **Object-Oriented Programming (OOP)** principles, modular logic, and secure data handling.

---

## 🛠️ Key Technical Implementations

### 1. Secure Banking System (`secure-banking-system.cpp`)

A financial management simulation focused on data protection and transaction integrity.
* **Encapsulation:** Used private access specifiers to protect sensitive account balances.
* **Flexibility:** Implemented Constructor Overloading to handle various account initialization scenarios.
* **Features:** Real-time balance updates and transaction logging.

---

### 2. Employee Salary & Payroll (`employee-salary-system.cpp`)

An automated system for institutional payroll processing.
* **Logic:** Engineered sophisticated conditional logic to calculate net pay, deductions, and bonuses based on role.
* **Validation:** Implemented rigorous input sanitization to prevent logical errors in financial calculations.

---

### 3. Vehicle Management Hierarchy (`vehicle-inheritance-system.cpp`)

A structural project demonstrating scalable code architecture.
* **Inheritance:** Modeled real-world relationships using base and derived classes to eliminate code redundancy.
* **Scalability:** Designed to be easily extended with new vehicle types without modifying existing core logic.

---

## 🚀 Core Competencies Demonstrated

* **The Four Pillars:** Proficient in Abstraction, Encapsulation, Inheritance, and Polymorphism.
* **Clean Code:** Standardized naming conventions (kebab-case) and clear vertical spacing for readability.
* **Documentation:** Comprehensive in-code comments and detailed README walkthroughs.
